유저에 대한 접근 레벨은 이전에 말한 것처럼 기능에 따라 4가지로 분류가 가능하고 접근성에 따라서는 크게 다음과 같이 3개로 나눌 수 있다.
1. view : 접근하고자 하는 데이터 영역에 대해서 조회가 가능, 하지만 편집이나 생성, 추출이나 삭제가 불가
2. create/ edit: 해당 데이터 영역에 접근해서 데이터를 생성하거나 편집이 가능, 하지만 해당 데이터 영역을 다른 곳으로 publish 하거나, 삭제하는 것은 불가능
3. publish/ delete: 해당 데이터 영역에 데이터 일부를 삭제하거나 테이블을 전부 삭제할 수도 있다. 데이터에 대한 생성이나 수정, 템플릿을 만드는 것은 불간으하다.
